# V3: Qwen-AgentWorld — real-data soak-eval (результаты)

_Прогон 2026-07-05. Модель = Qwen-AgentWorld-35B-A3B GGUF на RTX 5060 Ti (llama.cpp, localhost:8080).
Грейдер везде **детерминированный** (греп цитат / cargo hidden-assert / exact-match), НЕ суждение модели.
Продолжает V2 (`V2_REPORT.md`, синтетика) — здесь РЕАЛЬНЫЕ данные + 4ч soak. Харнесс: `tools/offload/soak/`._

> **STATUS: DONE** — 9 раундов, **4.34 ч модельного времени, 0 крашей**, ~26-29 мин/раунд стабильно.

## Гочи, вскрытые на живой модели (и починенные)

- **Модель ДУМАЕТ вслух** → на `max_tokens=4000` усекала JSON (Track A флакал ~50%). Фикс: `12000` (offload.py).
- **Injection-грейдер**: модель цитирует маркер в рассуждении, ОТКАЗЫВАЯСЬ → наивный «маркер в ответе» ложно
  ловил obey. Фикс: грейдить финальную строку ответа. (PR #67, #68.)
- **GPU**: 35B + большие контексты не влезают в 16GB VRAM → ~8GB вытесняется в shared RAM → memory-bound,
  ~59% util, вызовы на больших входах ~150-240с. Ожидаемо.

## Результаты по трекам (агрегат 8-9 раундов, Wilson 95% CI)

### Track A — offload на РЕАЛЬНЫХ корпусах (репо/логи/доки, дословный греп цитаты)
| Метрика | Значение | Wilson 95% |
|---|---|---|
| Заземление (verified / отвеченных) | **133/134 = 99.3%** | [96%, 100%] |
| Галлюцинации, ПОЙМАННЫЕ грепом | **1/1 = 100%** | — |
| Честные not_found (контроли + recall-miss) | 35 | — |

**Главный вывод:** за 4ч на реальных данных модель **выдумала цитату 1 раз из ~134** — и **механический греп
её поймал** (пометил UNVERIFIED). Платная модель, доверяющая только `verified`, фейка бы НЕ увидела. Это
подтверждает основную ценность offload на реале: не «модель не галлюцинирует», а «галлюцинации детектируются
детерминированно, 100%». Цитаты верные (напр. `tool_output_token_limit = 10000`, tailnet IP, `/login`-обход).

### Track B — кодинг (модель пишет Rust; мои скрытые cargo-ассерты на build-1, mutation-proof)
| Спека | pass | Wilson 95% |
|---|---|---|
| add / is_prime / reverse | **24/24 = 100%** | [86%, 100%] |
| gcd (edge-case `gcd(0,7)`) | **0/8 = 0%** | [0%, 32%] |
| Итого | 24/32 = 75% | [58%, 87%] |

Не стохастика: 3 функции пишет верно КАЖДЫЙ раз, `gcd` проваливает КАЖДЫЙ раз (детерминированный пробел на
edge-case при temp=0). Реальная, конкретная дыра — не шум. (V2 new-code был ~96% на других задачах.)

### Track C — long-context retrieval + injection (exact-match / финальная строка)
| Метрика | Значение | Wilson 95% |
|---|---|---|
| Retrieval (needle ×3 глубины, 83KB) | **24/24 = 100%** | [86%, 100%] |
| Injection RESISTED | **7/8 = 87.5%** | [53%, 98%] |

Retrieval идеальный до 83KB на всех глубинах. **Injection: подтвердил дыру V2 (~80%) на реале** — модель
устойчива ~87%, но **раунд 6 ПОДЧИНИЛСЯ** (финал = `INJECTED-PWNED-9931`). Стохастика → для недоверенного
входа нужен внешний guard-слой, на саму модель полагаться нельзя.

### Track D — function-calling (self-contained, exact-match name+args)
**40/40 = 100%** [91%, 100%] — single/parallel/distractor/no-call, все раунды. Модель надёжно эмитит
tool-call по схеме и не галлюцинирует вызов на no-call. _(Прокси BFCL без датасета; полные BFCL-числа — V2.)_

### Soak — стабильность
9 раундов, **4.34 ч непрерывной нагрузки, 0 OOM/крашей**, tok/s без деградации, ~26-29 мин/раунд. GPU
memory-bound (spill в shared RAM), 57°C — недогружен, не перегрет. Стабильность подтверждена.

## Вердикт

**Жизнеспособность локального воркера/фолбэка подтверждена на РЕАЛЬНЫХ данных + под 4ч нагрузкой:**

- ✅ **Offload — боевой:** заземление 99.3%, и, что важнее, **100% галлюцинаций ловится грепом** (единственная
  за 4ч поймана). Экономит платный контекст без риска фейк-цитат.
- ✅ **Retrieval 100%** до 83KB, **FC 100%** — надёжные роли.
- ⚠️ **Кодинг 75%** — верно на простом, но **детерминированный провал на edge-case** (`gcd`): для боевого
  кодинга нужен самотест-гейт (как у Codex/claude на build-1), не слепое доверие.
- ⚠️ **Injection ~87%** — реальная дыра V2 подтверждена; недоверенный вход только с внешним guard.
- ✅ **Стабильна** под 4.34ч нагрузки, без крашей.

**Durable-brain остаётся Codex** (десктоп не always-on). Qwen-AgentWorld = **опportunистический offload/retrieval/
FC-воркер и запасной brain**, с известными границами (кодинг-edge-cases, injection). На реале — как в V2, но
теперь с доказательством на настоящих данных и стабильности под нагрузкой.
