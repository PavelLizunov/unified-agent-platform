### Формат — VOID (не измеримо этими данными)

### think on vs off
Эквивалентность точности — только там, где рассуждение не нужно (заученный код, тривиальный FC, лексический NIAH).
На многошаговой задаче (агрегация) think-ON проиграл 68% сэмплов **из-за обрезки**, а не по существу. → «think off»
= защитимый выбор **латентности**; «нейтрально по точности для агента» — **не доказано** (см. Ф1).
