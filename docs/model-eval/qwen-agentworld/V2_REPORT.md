## Что это значит для north-star (vibe-coding)
Кандидат **жизнеспособен как кодинг/агент-brain**: агентный кодинг реально силён, retrieval по большому контексту
отличный, кодинг не заучен. **Два блокера до автономного продакшна:** (1) 🔴 инъекции — обязателен guardrail;
(2) само-верификация одна ненадёжна (ложные отклонения) → гейтом должны быть ИСПОЛНЯЕМЫЕ тесты (их модель гоняет и
чинит на 100%), а суждение модели — вторично. Reasoning-по-длинному-контексту — с осторожностью.
